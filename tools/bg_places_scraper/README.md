# BG Places Scraper

Помощен Python проект за извличане и подготовка на данни за places в България.

Цел:
- извличане на публични данни за населени места / туристически локации
- нормализиране на данните
- експортиране във формат, подходящ за Laravel import

Проектът не е част от Laravel приложението, а помощен инструмент, държан вътре в основното repo.
